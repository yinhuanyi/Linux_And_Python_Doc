Python 全栈开发文档¶

此开发文档，涵盖了Python开发的所有内容，包括: Python的基本数据类型、Python运算、Python控制流、Python模块导入、Python内置模块、Python函数、 Python设计思想、Python面向对象、Python高级模块部分 Python Socket 网络编程、前端开发（HTML、CSS、JS）、Juery框架、Bootstrap UI框架 Django框架、Redis缓存、MySQL关系数据库、MongoDB NoSQL数据库

同时也包含实战开发部分： 通过Python对日志清洗实现日志分析、基于ansible的DMDB二次开发、 基于Django框架在阿里云服务器上线教学管理系统、基于Django Rest Framework在阿里云服务器上线电商交易平台、 基于docker + gitlabCI + K8S 实现Python项目的自动化CI/CD
